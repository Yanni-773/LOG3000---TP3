# Static Directory

## Objectif
Ce répertoire contient tous les assets statiques de l'application web, notamment les fichiers CSS qui définissent l'apparence de l'interface utilisateur.

## Fichiers principaux
- `style.css`: Définit les styles visuels de la calculatrice, incluant :
  - La mise en page de la calculatrice
  - Le style des boutons
  - Les couleurs et le thème de l'interface
  - La réactivité pour différentes tailles d'écran
  
  Le fichier `style.css` est lié au fichier HTML principal de la calculatrice (`index.html`) pour appliquer les styles à l'interface utilisateur.

## Dépendances
- Aucune dépendance externe - utilise uniquement du CSS standard
- Compatible avec tous les navigateurs modernes

## Notes pour les développeurs
- Les modifications de style doivent maintenir la cohérence visuelle
- Suivre les conventions BEM pour le nommage des classes CSS
- Tester les changements sur différents navigateurs et tailles d'écran
